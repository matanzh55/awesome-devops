# Selefra

![Selefra Logo](https://avatars.githubusercontent.com/u/109767323?s=200&v=4)

## Overview

Selefra is an open-source policy-as-code software that provides analytics for multi-cloud and SaaS. It allows you to write SQL queries to analyze your cloud infrastructure and enforce policies across multiple cloud providers.

## Key Features

- Multi-cloud analytics with SQL queries
- Policy as code for infrastructure compliance
- Support for AWS, Azure, GCP, and other cloud providers
- Open-source and extensible

## Getting Started

Install Selefra using pip or from source. Define your cloud connections and start writing SQL queries to analyze your infrastructure.
