# Spacelift

![Spacelift Logo](https://avatars.githubusercontent.com/u/55009436?s=200&v=4)

## Overview

Spacelift is a flexible orchestration solution for Infrastructure as Code development. It provides a drop-in Terraform Cloud alternative with usage-based pricing, unlimited concurrency, and native GitHub integration.

## Key Features

- Drop-in replacement for Terraform Cloud
- Usage-based pricing model
- Unlimited concurrency for runs
- Native GitHub integration
- Support for Terraform, Pulumi, and other IaC tools

## Getting Started

Sign up for a Spacelift account and connect your GitHub repository. Configure your stacks and start deploying infrastructure.
