# Atlantis

![Atlantis Logo](https://avatars.githubusercontent.com/u/2391026?s=200&v=4)

## Overview

Atlantis is a Terraform Pull Request Automation tool. It runs terraform plan and apply commands on pull requests, allowing for automated infrastructure code reviews and deployments.

## Key Features

- Automated terraform plan on pull requests
- Apply infrastructure changes with pull request reviews
- Support for GitHub, GitLab, and Bitbucket
- Locking mechanism to prevent concurrent modifications
- Customizable workflows and policies

## Getting Started

Install Atlantis on your infrastructure or use the hosted version. Configure webhooks in your Git repository to trigger Atlantis on pull requests.
