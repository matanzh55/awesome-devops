# Stacktape

![Stacktape Logo](https://avatars.githubusercontent.com/u/80503481?s=200&v=4)

## Overview

Stacktape is a developer-friendly Infrastructure as Code framework built on top of AWS. It simplifies deploying and managing cloud infrastructure with a focus on developer experience and productivity.

## Key Features

- Simplified AWS infrastructure deployment
- Built-in best practices and security
- Support for multiple programming languages
- Local development and testing capabilities
- Cost optimization features

## Getting Started

Install the Stacktape CLI and create a stacktape.yml configuration file. Use the deploy command to provision your infrastructure on AWS.
