# Scaleway

![Scaleway Logo](../logos/scaleway.svg)

## Overview

Scaleway is a cloud computing platform that provides a single way to create, deploy, and scale infrastructure in the cloud. It offers a range of services including compute instances, storage, and networking solutions designed for developers and businesses.

## Key Features

- Flexible compute instances with various CPU and GPU options
- Object storage and block storage solutions
- Managed databases and Kubernetes services
- Global data center locations for low-latency access

## Getting Started

To get started with Scaleway, sign up for an account at scaleway.com and use the console or API to provision resources. You can also use tools like Terraform for infrastructure as code.
