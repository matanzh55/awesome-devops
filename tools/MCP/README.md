# MCP (Model Context Protocol)

Model Context Protocol servers that extend AI capabilities with external data sources and tools.

## Overview

The Model Context Protocol (MCP) is a standard for connecting AI assistants to external tools and data sources. MCP servers provide various functionalities like database access, web scraping, file system operations, and more.

## MCP Servers

- [1Panel](1Panel/) - Modern, open-source Linux server operation and management panel
- [activepieces](activepieces/) - No-code workflow automation platform
- [context7](context7/) - Context-aware AI assistant for developers
- [fastmcp](fastmcp/) - Fast and lightweight MCP server for building AI agents
- [Figma-Context-MCP](Figma-Context-MCP/) - Figma design integration for AI assistants
- [genai-toolbox](genai-toolbox/) - Google Cloud GenAI toolbox for AI model access
- [github-mcp-server](github-mcp-server/) - GitHub integration for AI assistants
- [mindsdb](mindsdb/) - AI-powered database management and analytics
- [playwright-mcp](playwright-mcp/) - Browser automation and web scraping with Playwright
- [serena](serena/) - AI-powered development assistant
