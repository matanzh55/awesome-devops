# genai-toolbox

Google Cloud GenAI Toolbox MCP server

## Key Features

- Integration with Google Cloud AI
- Generative AI model access
- Vertex AI support
- Multi-modal capabilities

## Getting Started

```bash
# Install genai-toolbox
pip install google-cloud-genai-toolbox

# Set up authentication
gcloud auth application-default login

# Configure project
export GOOGLE_CLOUD_PROJECT=your-project-id

# Run the toolbox
genai-toolbox
```
