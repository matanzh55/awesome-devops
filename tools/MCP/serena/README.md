# serena

Serena MCP server for AI-powered development

## Key Features

- AI-assisted coding
- Code generation and completion
- Multi-language support
- Integration with popular IDEs

## Getting Started

```bash
# Install serena
npm install -g @serena-rx/cli

# Configure API key
serena config --api-key YOUR_API_KEY

# Start coding assistance
serena start
```
