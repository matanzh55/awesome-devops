# Figma-Context-MCP

Figma Context MCP server for design integration

## Key Features

- Access Figma designs and components
- Extract design tokens and styles
- Generate code from designs
- Real-time design collaboration

## Getting Started

```bash
# Install Figma-Context-MCP
npm install -g @yuhang/figma-context-mcp

# Configure Figma access token
export FIGMA_ACCESS_TOKEN=your_token_here

# Start the server
figma-context-mcp
```
