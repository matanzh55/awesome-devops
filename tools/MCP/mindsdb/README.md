# mindsdb

Federated query engine for AI - The only MCP Server you'll ever need

## Key Features

- Federated query engine for AI
- Integrates with various data sources
- Machine learning capabilities
- AI-powered data analysis

## Getting Started

```bash
# Install mindsdb
pip install mindsdb

# Start the server
mindsdb --api=mcp
```
