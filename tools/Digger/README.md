# Digger

![Digger Logo](https://avatars.githubusercontent.com/u/107161916?s=200&v=4)

## Overview

Digger is an open-source Infrastructure as Code management tool that runs within your CI/CD system. It provides a collaborative way to manage Terraform deployments with pull request-based workflows.

## Key Features

- Pull request-based Terraform deployments
- Integration with GitHub, GitLab, and Bitbucket
- Automatic drift detection
- Policy enforcement and compliance checks
- Support for multiple cloud providers

## Getting Started

Install Digger in your CI/CD pipeline. Configure digger.yml in your repository and start managing infrastructure changes through pull requests.
