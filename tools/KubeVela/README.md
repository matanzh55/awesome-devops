# KubeVela

![KubeVela Logo](https://avatars.githubusercontent.com/u/55775339?s=200&v=4)

## Overview

KubeVela is a modern application delivery platform that makes deploying and operating applications across today's hybrid, multi-cloud environments easier, faster and more reliable. It provides a developer-friendly abstraction layer over Kubernetes.

## Key Features

- Application-centric deployment model
- Support for hybrid and multi-cloud environments
- Built-in application lifecycle management
- Extensible through CUE language
- Integration with existing Kubernetes tools

## Getting Started

Install KubeVela using vela CLI. Define your applications using vela.yaml files and deploy them to Kubernetes clusters.
