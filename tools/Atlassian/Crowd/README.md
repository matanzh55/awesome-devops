# Crowd

![Crowd Logo](https://wac-cdn.atlassian.com/dam/jcr:crowd-logo.png)

## Overview

Crowd is Atlassian's identity management tool for on-premises deployments providing SSO, user directories, and centralized authentication for Atlassian products.

## Key Features

- Single sign-on for Atlassian Server/Data Center
- Centralized user directory and group management
- Integration with LDAP and Active Directory

## Getting Started

```bash
# Install Crowd (Server)
wget https://www.atlassian.com/software/crowd/downloads/crowd-x.x.x.tar.gz
tar -xzf crowd-x.x.x.tar.gz
cd crowd-x.x.x/bin
./start-crowd.sh

# Configure directories and applications via the web UI
```
