# Crucible

![Crucible Logo](https://wac-cdn.atlassian.com/dam/jcr:crucible-logo.png)

## Overview

Crucible is a collaborative code review tool focused on pre-commit and post-commit reviews with threaded comments and audit trails.

## Key Features

- Peer code review workflows
- Inline comments and threaded discussions
- Audit trails and metrics
- Integration with Fisheye and other SCMs

## Getting Started

```bash
# Install Crucible (Server)
wget https://www.atlassian.com/software/crucible/downloads/crucible-x.x.x.tar.gz
tar -xzf crucible-x.x.x.tar.gz
cd crucible-x.x.x/bin
./start-crucible.sh

# Create a review via UI or API
```
