# Forge

![Forge Logo](https://wac-cdn.atlassian.com/dam/jcr:forge-logo.png)

## Overview

Forge is Atlassian's serverless platform for building cloud-native apps for Atlassian Cloud products with simplified deployment, storage, and permissions.

## Key Features

- Serverless functions and storage
- Simplified app lifecycle and deployment
- Permissions and product context
- CLI tooling for development and deployment

## Getting Started

```bash
# Install the Forge CLI
npm install -g @forge/cli

# Login and create a new app
forge login
forge create

# Deploy
forge deploy
forge install
```
