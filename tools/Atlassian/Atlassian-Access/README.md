# Atlassian Access

![Atlassian Access Logo](https://wac-cdn.atlassian.com/dam/jcr:access-logo.png)

## Overview

Atlassian Access provides enterprise-grade security controls for Atlassian Cloud products including SSO via SAML, SCIM user provisioning, and organization-wide policies.

## Key Features

- Single sign-on (SAML) across Atlassian Cloud
- SCIM provisioning and deprovisioning
- Enforced security policies and 2FA
- Audit logs and organization controls
