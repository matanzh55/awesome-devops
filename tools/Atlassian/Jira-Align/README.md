# Jira Align

![Jira Align Logo](https://wac-cdn.atlassian.com/dam/jcr:jira-align-logo.png)

## Overview

Jira Align is an enterprise planning solution that connects business strategy to technical work, enabling portfolio management, roadmapping, and scaled agile practices.

## Key Features

- Portfolio and program management
- Strategy alignment and roadmapping
- PI planning and capacity planning
- Cross-team visibility and reporting
