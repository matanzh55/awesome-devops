# Sourcetree

![Sourcetree Logo](../../logos/sourcetree.svg)

## Overview

Sourcetree is a free Git and Mercurial desktop client for Windows and macOS that simplifies repo management with a visual interface for commits, branching, and history.

## Key Features

- Visual commit history and branch management
- Support for Git and Mercurial
- Integration with Bitbucket and GitHub
- Staging/unstaging UI and interactive rebase support

## Getting Started

```bash
# Download and install Sourcetree from Atlassian
# https://www.sourcetreeapp.com/

# Clone a repository via GUI or command-line
git clone https://bitbucket.org/{workspace}/{repo_slug}.git
```
