# Halp

![Halp Logo](https://wac-cdn.atlassian.com/dam/jcr:halp-logo.png)

## Overview

Halp provides conversational ticketing inside Slack and Microsoft Teams, allowing teams to create and manage tickets from chats.

## Key Features

- Create tickets from chat messages
- Workflow automation and routing
- Integration with Jira and other tools
