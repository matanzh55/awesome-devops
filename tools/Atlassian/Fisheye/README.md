# Fisheye

![Fisheye Logo](https://wac-cdn.atlassian.com/dam/jcr:fisheye-logo.png)

## Overview

Fisheye provides code search, visualization, and repository analytics for Git, Mercurial, Subversion and Perforce, often paired with Crucible for code review workflows.

## Key Features

- Repository browsing and search
- Change and commit visualization
- Integration with Crucible for code reviews

## Getting Started

```bash
# Install Fisheye (Server)
wget https://www.atlassian.com/software/fisheye/downloads/fisheye-x.x.x.tar.gz
tar -xzf fisheye-x.x.x.tar.gz
cd fisheye-x.x.x/bin
./start-fisheye.sh

# Add a repository via the web UI
```
