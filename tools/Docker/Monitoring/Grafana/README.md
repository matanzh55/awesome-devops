# Grafana

![Grafana Logo](../../../logos/grafana.svg)

## Overview

Grafana is an open-source analytics and visualization platform for metrics, logs, and traces. It integrates with Prometheus, InfluxDB, Elasticsearch and many other data sources to build dashboards and alerts.

## Key Features

- Rich dashboarding and visualization panels
- Multiple data source support and plugins
- Alerting and notification channels
- Provisioning and templating for reproducible dashboards

## Getting Started

```bash
# Run Grafana with Docker
docker run -d -p 3000:3000 --name=grafana grafana/grafana

# Open http://localhost:3000 and login (default admin/admin)
```
