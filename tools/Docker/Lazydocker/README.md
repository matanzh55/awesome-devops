# Lazydocker

![Lazydocker Logo](https://raw.githubusercontent.com/jesseduffield/lazydocker/master/.github/brand.png)

## Overview

Lazydocker is a simple terminal UI for both Docker and Docker Compose designed to simplify container management during local development. It provides an intuitive TUI for viewing containers, logs, and managing services.

## Key Features

- Interactive TUI for containers, images, volumes, and networks
- View and follow logs, inspect containers, and execute commands
- Cross-platform and lightweight binary distribution

## Getting Started

```bash
# Install via Homebrew (macOS)
brew install jesseduffield/lazydocker/lazydocker

# Run in a project with docker-compose.yml
lazydocker
```
