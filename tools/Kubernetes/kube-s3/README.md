# kube-s3

Kubernetes pods used shared S3 storage

## Key Features

- Shared S3 storage for Kubernetes pods
- Easy configuration and deployment
- Scalable and reliable storage solution
- Integration with Kubernetes secrets

## Getting Started

```bash
# Deploy kube-s3
kubectl apply -f https://raw.githubusercontent.com/freegroup/kube-s3/master/kube-s3.yaml

# Usage
# Configure your pods to use the S3 storage
```
