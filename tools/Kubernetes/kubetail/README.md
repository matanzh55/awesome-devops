# kubetail

Bash script to tail Kubernetes logs from multiple pods at the same time

## Key Features

- Tail logs from multiple pods simultaneously
- Filter logs by labels or namespaces
- Real-time log streaming with color coding
- Easy to use command-line interface

## Getting Started

```bash
# Install kubetail
curl -s https://raw.githubusercontent.com/johanhaleby/kubetail/master/install.sh | bash

# Usage example
kubetail -l app=myapp
```
