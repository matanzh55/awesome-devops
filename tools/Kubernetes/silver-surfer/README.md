# silver-surfer

Check ApiVersion compatibility and provide Migration path for Kubernetes objects when upgrading Kubernetes to latest versions

## Key Features

- API version compatibility checking
- Migration path suggestions
- Upgrade planning assistance
- Automated analysis of Kubernetes objects

## Getting Started

```bash
# Deploy silver-surfer
kubectl apply -f https://raw.githubusercontent.com/devtron-labs/silver-surfer/master/deploy/silver-surfer.yaml

# Run analysis
kubectl silver-surfer
```
