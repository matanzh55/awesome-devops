# katafygio

Dump, or continuously backup Kubernetes objects as yaml files in git

## Key Features

- Backup Kubernetes objects to git
- YAML file dumps
- Continuous backup options
- Version control for cluster state

## Getting Started

```bash
# Deploy katafygio
kubectl apply -f https://raw.githubusercontent.com/bpineau/katafygio/master/deploy/katafygio.yaml

# Configure git repo
```
