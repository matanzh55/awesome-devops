# Monokle

Desktop unified visual tool for authoring, analysis and deployment of Kubernetes configurations

## Key Features

- Visual authoring of Kubernetes manifests
- Analysis and validation of configurations
- Deployment capabilities
- User-friendly desktop interface

## Getting Started

```bash
# Download from the official website
# https://monokle.io/

# Or install via brew
brew install monokle

# Usage
# Open the application and load your Kubernetes configs
```
