# kube-shell

Kubernetes shell: An integrated shell for working with the Kubernetes

## Key Features

- Interactive shell for Kubernetes commands
- Auto-completion for kubectl commands
- Syntax highlighting
- Command history
- Integration with kubectl

## Getting Started

```bash
# Install kube-shell
pip install kube-shell

# Run kube-shell
kube-shell
```
