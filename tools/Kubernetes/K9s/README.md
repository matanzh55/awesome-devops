# K9s

Kubernetes CLI To Manage Your Clusters In Style!

## Key Features

- Terminal-based UI for Kubernetes clusters
- Real-time monitoring of resources
- Easy navigation and management of pods, services, and other Kubernetes objects
- Support for multiple clusters and contexts
- Plugin system for extending functionality

## Getting Started

```bash
# Install k9s using Homebrew (macOS)
brew install k9s

# Or using other package managers
# For Linux
curl -L https://github.com/derailed/k9s/releases/latest/download/k9s_Linux_amd64.tar.gz | tar xz
sudo mv k9s /usr/local/bin/

# Run k9s
k9s
```
