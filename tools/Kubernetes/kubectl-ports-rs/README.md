# kubectl-ports-rs

A kubectl krew plugin to provide a list of exposed ports on kubernetes resources

## Key Features

- List exposed ports on Kubernetes resources
- kubectl plugin for easy integration
- Fast and efficient port scanning
- Supports various resource types

## Getting Started

```bash
# Install kubectl krew if not installed
# Then install the plugin
kubectl krew install ports-rs

# Usage
kubectl ports-rs
```
