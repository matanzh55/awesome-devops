# Kubeshark

TCPDump and Wireshark re-invented for Kubernetes

## Key Features

- Real-time visibility into Kubernetes traffic
- Service Map for understanding microservice architecture
- Query language for filtering and analyzing traffic
- Integrations with popular tools
- API traffic recording and replay

## Getting Started

```bash
# Install kubeshark
curl -Lo kubeshark https://github.com/kubeshark/kubeshark/releases/latest/download/kubeshark_linux_amd64
chmod +x kubeshark
sudo mv kubeshark /usr/local/bin/

# Run kubeshark
kubeshark tap
```
