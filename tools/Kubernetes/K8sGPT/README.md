# K8sGPT

Giving Kubernetes SRE superpowers to everyone

## Key Features

- AI-powered Kubernetes diagnostics
- Natural language queries for troubleshooting
- Integration with popular Kubernetes tools
- Automated issue detection and recommendations
- Support for multiple AI providers

## Getting Started

```bash
# Install k8sgpt
curl -Lo k8sgpt.tar.gz https://github.com/k8sgpt-ai/k8sgpt/releases/latest/download/k8sgpt_amd64.tar.gz
tar -xzf k8sgpt.tar.gz
sudo mv k8sgpt /usr/local/bin/

# Set up OpenAI API key
export OPENAI_API_KEY=your-api-key

# Analyze cluster
k8sgpt analyze
```
