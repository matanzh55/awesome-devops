# K8s-Insider

A zero-config way to access your Kubernetes cluster network

## Key Features

- Zero-configuration network access
- Direct access to cluster services
- Simplified networking for development
- Automatic proxy setup
- Secure tunnel creation

## Getting Started

```bash
# Install k8s-insider
go install github.com/TrueGoric/k8s-insider@latest

# Start the proxy
k8s-insider proxy
```
