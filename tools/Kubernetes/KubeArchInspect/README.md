# KubeArchInspect

Kubernetes Architecture Inspector

## Key Features

- Inspect Kubernetes cluster architecture
- Analyze cluster components and configurations
- Generate architecture diagrams
- Identify potential issues
- Support for ARM-based clusters

## Getting Started

```bash
# Install kube-arch-inspect
# Follow installation instructions from the repository

# Run inspection
kube-arch-inspect analyze
```
